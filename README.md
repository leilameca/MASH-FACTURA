# MASH Flow

Sistema administrativo premium para MASH / Martinez Star Home.

## Comandos

```bash
npm install
npm run dev
```

## Variables de entorno

Crea un archivo `.env` basado en `.env.example`:

```bash
VITE_SUPABASE_URL=https://your-project.supabase.co
VITE_SUPABASE_ANON_KEY=your-anon-key
```

## Stack

- React + Vite
- Tailwind CSS
- React Router
- Lucide React
- Supabase Auth/DB/Storage
- React Hook Form
- @react-pdf/renderer
